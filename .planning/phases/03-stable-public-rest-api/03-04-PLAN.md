<objective>
Fix the curator proposal form so ALL submissions — regardless of submitter role — create a pending proposal record that requires a separate admin approval step before the mapping appears in the database.

Purpose: UAT Test 7 failed because /submit in api.py called create_mapping() directly, bypassing the proposal workflow. Approved mappings had null provenance (approved_by, approved_at, suggestion_score). The correct flow is: submit -> pending proposal -> admin approves -> mapping created with provenance.

Output: Modified models.py (migration + new method), modified api.py (/submit creates proposal), modified admin.py (approve_proposal handles new-pair), new test coverage in test_app.py.
</objective>

**B. src/blueprints/admin.py — handle new-pair proposals in approve_proposal()**

In approve_proposal() (around lines 262-283), the current code reads `mapping_id = proposal["mapping_id"]` then branches on proposed_delete. Add a branch for new-pair proposals (mapping_id is None):

Replace the block starting at `# Apply the proposed changes` with:

```python
# Apply the proposed changes
success = True
mapping_id = proposal["mapping_id"]

if proposal["proposed_delete"]:
    # Delete the mapping
    success = mapping_model.delete_mapping(mapping_id, admin_username)
    action = "deleted"
elif mapping_id is None:
    # New-pair proposal: create the mapping then write provenance
    approved_at = datetime.utcnow().isoformat()
    proposal_score = proposal.get("suggestion_score")
    new_mapping_id = mapping_model.create_mapping(
        ke_id=proposal["ke_id"],
        ke_title=proposal["ke_title"],
        wp_id=proposal["wp_id"],
        wp_title=proposal["wp_title"],
        connection_type=proposal.get("new_pair_connection_type") or proposal.get("proposed_connection_type"),
        confidence_level=proposal.get("new_pair_confidence_level") or proposal.get("proposed_confidence"),
        created_by=proposal.get("github_username") or admin_username,
    )
    if new_mapping_id:
        success = mapping_model.update_mapping(
            mapping_id=new_mapping_id,
            approved_by_curator=admin_username,
            approved_at_curator=approved_at,
            suggestion_score=proposal_score,
        )
    else:
        success = False
    action = "created"
else:
    # Existing mapping revision proposal: update the mapping with provenance
    approved_at = datetime.utcnow().isoformat()
    proposal_score = proposal.get("suggestion_score")
    success = mapping_model.update_mapping(
        mapping_id=mapping_id,
        connection_type=proposal["proposed_connection_type"],
        confidence_level=proposal["proposed_confidence"],
        updated_by=admin_username,
        approved_by_curator=admin_username,
        approved_at_curator=approved_at,
        suggestion_score=proposal_score,
    )
    action = "updated"
```

        )

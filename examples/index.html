üìö Quick API Reference

Key Endpoints

  • GET /api/v1/mappings - List mappings with pagination and filtering
  • GET /api/v1/mappings/{id} - Get specific mapping
  • GET /api/v1/mappings/stats - Get dataset statistics
  • GET /export/{format} - Export data in various formats
  • GET /dataset/metadata - Get comprehensive metadata

Base URL: https://ke-wp-mapping.org

Rate Limits: 1000 requests/hour (general), 100 requests/hour (submissions)
